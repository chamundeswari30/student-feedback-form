# 🎓 Student Feedback System

## 📌 Project Description

The **Student Feedback System** is a simple web-based application developed using **HTML and CSS**. It allows students to register and provide structured feedback about courses and instructors through a two-step questionnaire.

This project is designed as a mini project for the subject **Essential Practices of Software Development**.

---

## 🚀 Features

* 🏠 Home page with navigation
* 📝 Student registration form
* 📊 Feedback form divided into two sections (1–10 and 11–20 questions)
* ✅ Multiple-choice answers (Excellent, Good, Average, Poor)
* 🎉 Thank you page after submission
* 🎨 Modern UI with colourful sea-blue theme
* 💻 Built using only HTML and CSS (no JavaScript)

---

## 🗂️ Project Structure

```
Student-Feedback-System/
│
├── index.html        # Home page
├── register.html     # Registration form
├── feedback1.html    # Feedback questions (1–10)
├── feedback2.html    # Feedback questions (11–20)
└── thankyou.html     # Final thank you page
```

---

## ⚙️ How to Run the Project

1. Download or copy all files into one folder
2. Open the folder
3. Double-click on `index.html`
4. The project will open in your web browser
